### @SafeVarargs
* java1.7
* Varargs가 생성자나 method에서 적용이 될 때, unchecked warnings 검사를 잠재적으로 실행하지 않습니다. Annotation type으로 사용될 때는 varargs에 관련 된 unchecked warnings를 발생시키지 않습니다.
