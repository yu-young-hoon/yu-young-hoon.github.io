SBT Simple Build Tool의 약자로 Maven이나 Gradle처럼 프로젝트를 셋팅하고 빌드를 관리해주는 툴입니다.

| Command     | Notes                                |
|-------------|--------------------------------------|
| sbt help    | sbt 명령어 일람과 도움말                  |
| sbt compile | build.sbt를 바탕으로 프로젝트를 빌드 합니다.  |
| sbt run     | 프로젝트를 빌드하고 실행합니다.              |
| sbt clean   | 프로젝트에서 생성된 모든 파일을 제거합니다.     |
